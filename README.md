## Tictactoe

A simple web tictactoe game made with [ReactJS][reactjs] and [Redux][redux].
[Play online here][play].



### Building locally

Prerequisites:

- NodeJS
- NPM
- [Webpack][webpack] (installed globally)


```bash
$ git clone https://github.com/atisawd/tictactoe.git
$ cd tictactoe
$ npm install
$ npm run dev
```
